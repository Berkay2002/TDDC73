## Component Categories

### Visual Components
CustomCard, CustomSlider, CustomCircularProgress, and CustomToggleSwitch demonstrate how to use `CustomPaint` and `Canvas` to create custom visual elements with shadows, colors, and animations.

### Layout Components
VStack, HStack, and ZStack show how to implement custom layout logic using `RenderBox`, constraint-based sizing, and parent data for advanced positioning and flexibility.
